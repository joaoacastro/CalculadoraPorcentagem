# Calculadora de Porcentagens

Este é um projeto simples em HTML, CSS e JavaScript para calcular porcentagens de diversas maneiras.

## Funcionalidades

- Quanto é ___% de ___?
- O valor ___ é qual a porcentagem de ___?
- Um valor de ____ que AUMENTOU para ____. Qual foi o aumento percentual?
- Um valor de ____ que DIMINUIU para ____. Qual foi a diminuição percentual?
- O valor ___ sobre o valor ___ é quantos por cento?
- Tenho um valor de ___ e quero AUMENTAR ___%. Qual é o resultado?
- Tenho um valor de ___ e quero DIMINUIR ___%. Qual é o resultado?
- Tenho um valor inicial que AUMENTOU em ___% e passou para ___ . Qual é o valor inicial?
- Tenho um valor inicial que DIMINUIU em ___% e passou para ___ . Qual é o valor inicial?

## Como usar

1. Abra o arquivo `index.html` em um navegador web.
2. Preencha os campos de entrada para o cálculo desejado.
3. Clique no botão "Calcular".
4. O resultado aparecerá ao lado do botão.

## Arquivos

- `index.html`: Estrutura da página.
- `styles.css`: Estilos da página.
- `script.js`: Lógica de cálculo em JavaScript.
